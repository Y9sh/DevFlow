# DevFlow

A Python-based command-line utility built to simplify parts of my local development workflow, including Git management, running local tools/services, and experimenting with AI-assisted development workflows.

## Features
- Initialize and manage Git repositories
- Automate commits and repository setup
- Run local development tools and services
- Integrate AI-assisted tooling workflows
- Execute terminal/system commands through Python

## Notes
This project was built as part of my learning and experimentation process in Python automation and development tooling.

## External setup 
1. LM Studio.
2. Aider installed globally
3. Excalocal (Node.js dependencies)

## Installation

1. Ensure you have Python 3.9+ installed.
2. Clone this project.
3. pip install -r requirements.txt (Python dependencies)
4. npm install (Node.js dependencies)
