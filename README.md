# Git Aider

A simple CLI tool to help you initialize a Git repository, create a `.gitignore` file, and run Aider with a selected model.

## Features

- Initialize a new Git repository.
- Create a `.gitignore` file with common project files.
- Commit changes with a custom message.
- Run Aider with a selected OpenAI model.
- Check for uncommitted changes.
- Update `.gitignore` with additional ignored files.
